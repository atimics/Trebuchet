# .launch File Format

Trebuchet consumes `.launch` files — a zip-based token launch format defined
in its own repository at [cenetex/launch](https://github.com/cenetex/launch).

The spec is at [SPEC.md](https://github.com/cenetex/launch/blob/main/SPEC.md).
License is MIT-0 — no attribution required.

See the standalone repo for the full format specification, JSON schemas,
encryption support, and extension points.
